# 開發環境
:::::::::::::::::::::::::::::::::::::::::::::::::::::::
:::: 
:::: MacOS Ventura v13.5.1
:::: Node.js v18.17.1
::::
::::::::::::::::::::::::::::::::::::::::::::::::::::::: 

## 安裝 truffle
::::::::::::::::::::::::::::::::::::::::::::::::::::::::
::::
::::   $ npm install -g truffle＠5.11.2
::::
::::::::::::::::::::::::::::::::::::::::::::::::::::::::

## 安裝 Ganache GUI / Ganache CLI
- 官方網址：https://archive.trufflesuite.com/ganache/
- 確保啟用 Ganache 
  - 如果是 MacOs 就開啟 Ganache Application, 選擇 QuickStart

## 確認 Social_Recovery/ 中的 truffle-config.js 正確設置，以便能連線 Ganache
  - Ganache 的預設是 host: "127.0.0.1" port: 7545

## 確認 console 是導向正確的路徑
::::::::::::::::::::::::::::::::::::::::::::::::::::::::
::::
::::  $ cd blockchain_hw8/Social_Recovery/
::::
::::::::::::::::::::::::::::::::::::::::::::::::::::::::

## 部署 TokenAuction.sol 智能合約
::::::::::::::::::::::::::::::::::::::::::::::::::::::::
::::
::::  $ truffle migrate --network development
::::
::::::::::::::::::::::::::::::::::::::::::::::::::::::::

## 測試智能合約
